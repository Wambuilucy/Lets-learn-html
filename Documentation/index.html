Colors Change

It is very simple to use different colors in text. Just add color class to desired text and enjoy our beautifull colorscheme.

It is very simple to change the colors

  
	<link rel="stylesheet" id="color" href="css/default.css">
  
 	for example if you need Blue Color just Replace (default.css) in (colors/blue.css)
  	
    	<link rel="stylesheet" id="color" href="css/colors/blue.css">
